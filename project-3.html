Controller Design for Crazyflie 2.0 Aerial Robot

Sliding mode controller designed for the CrazyFlie 2.0 drone to track desired trajectories. Implemented the controller in a simulation model of the drone in Gazebo with parameter tuning.

Demo Video

Project Overview

This project focuses on designing a sliding mode controller for altitude and attitude control of the Crazyflie 2.0 aerial robot to achieve trajectory tracking and waypoint navigation. The controller was designed to track desired trajectories by selecting a sliding surface s, ensuring the system state reaches and remains on this surface through the control input. MATLAB, Python, ROS, and Gazebo were used to develop and test the controller in simulation.

After parameter tuning, the controller achieved trajectory convergence, enabling the quadrotor to follow desired paths accurately. Challenges included extensive tuning, where parameters were adjusted iteratively to observe their effects and optimize performance. This work demonstrates the effectiveness of sliding mode control in ensuring robust trajectory tracking for micro aerial vehicles.

This project is private and thus the repository is not accessible publicly.
