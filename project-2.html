Teleoperation of Aerial Robots using an Augmented Reality Virtual Surrogate

Designed an AR interface using Unity and AR Foundations to control a drone via a virtual surrogate. Conducted user study showing a 95% preference for the surrogate interface over direct drone teleoperation.

Project Overview

This project explores the teleoperation of aerial robots by creating an augmented reality (AR) interface featuring a virtual surrogate. Developed in Unity using C#, the interface includes a drone, its virtual surrogate, and interactive obstacles in the form of rings. Both the drone and its virtual surrogate can be controlled via a gaming controller, providing an intuitive and immersive user experience.

A user study was conducted where participants navigated the drone and its virtual surrogate through the obstacles, and performance was evaluated using completion time metrics. Additionally, qualitative feedback and a 7-point Likert scale questionnaire were employed to assess user experience. This project demonstrates the potential of AR interfaces in improving teleoperation precision and user engagement for aerial robots.
