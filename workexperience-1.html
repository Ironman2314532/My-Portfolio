Robotics Engineer Intern, Void Robotics

Developed a custom SLAM algorithm using ROS2 and Nav2, integrating RTK GPS and ZED2 stereo cameras. Debugged and optimized robotic systems, ensuring seamless sensor integration and improved localization accuracy.

Project Overview

As a robotics engineer intern at Company X, I contributed to the enhancement of a high-payload autonomous robot by refining its navigation and localization systems. My primary focus was debugging and optimizing the custom SLAM algorithm, utilizing ROS2 and Nav2 frameworks. This involved integrating Real-Time Kinematic (RTK) GPS and ZED2 stereo cameras to improve the robot's accuracy in dynamic environments. I worked on fine-tuning the sensor fusion pipeline to ensure robust localization and seamless sensor integration, which led to improved overall system performance.

Additionally, I assisted in troubleshooting and debugging the company’s app for monitoring and controlling the robot. I frequently collaborated with version control using Git to manage code changes, and ensured that all components of the system, from the robot’s software stack to the app interface, were well-synchronized and optimized for performance. This hands-on experience enhanced my understanding of both the hardware and software aspects of autonomous robotic systems.
